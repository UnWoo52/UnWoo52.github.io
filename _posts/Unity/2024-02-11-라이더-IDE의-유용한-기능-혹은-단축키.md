# 개요

공유하면 좋은 Rider의 단축키 혹은 기능들

많은 사람들이 알거나 익숙한 단축키가 아닌 것들을 적어봄

생각날 때 마다 추가 중


# 단축키

ctrl + shift + F9 : 모든 디버그 중단점 일괄 해제

# 유용한 기능

탭이 많이 열려있을 때, Alt를 누른 상태로 탭 닫기를 누르기 : 해당 탭을 제외한 모든 탭이 닫힌다.
